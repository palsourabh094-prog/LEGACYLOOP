05 / Architecture

Built like plant equipment: inspectable, isolated, logged.

No black boxes. Every answer can be traced from client to source paragraph.

Hybrid Retrieval

BM25 keyword search catches exact tag numbers and clause references; dense embeddings catch meaning. Results are fused and re-ranked before the agent sees them. Equipment tags are first-class entities, not tokens.

Grounded Reasoning

The agent may only answer from retrieved evidence. Every sentence carries a document, page and paragraph reference. Below a confidence threshold, it declines and routes the query to a named senior engineer.

Air-Gapped Deployment

The full stack — parser, graph, vector store, model — runs inside the plant network or a private VPC. No document, query or answer ever crosses the fence. The audit log records all of it.
